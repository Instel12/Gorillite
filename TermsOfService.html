Terms of Service

Behavioral Rules

By running Gorillite VR, players should not:

  • Engage in any form of harassment or hate speech
  • Disrespect other players and the virtual environment
  • Modify the game's code in any way that doesnt comply with the Modding Rules
  • Engage in illegal activities
  • Manage an account under the age of 13 years old

Modding Rules

The term "modded" refers to a modified version of the game or a patch for the game, "mod" means modification and "modding" refers to the process of creating or using such modifications.

Gorillite will not help with any modifications to the game and account support will not be provided for modding reasons.

Please comply with the following rules while modding:

  • Do not distribute modified versions of the game
  • If the mod contains gameplay modifications, only make them apply if the game mode contains the string "modded"
  • Do not create mods that interact with any of the networking systems in the game

Not complying with these rules will result in a ban in the game's multiplayer services.
